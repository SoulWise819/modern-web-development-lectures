# WEEK3주차 2회차 Vue 강의 내용

본 문서는 Vue.js 학습 계획서를 수강생용과 강사용으로 분리해 관리하기 위한 개요 문서이다.

이번 스터디의 핵심 방향은 Todo List 앱을 단계적으로 완성하면서 Vue 핵심 기능을 자연스럽게 익히는 것이다. 문법을 한 번에 많이 설명하기보다, 기능 구현과 개념 설명을 반복적으로 연결하는 흐름을 유지한다.

## 문서 구성

- 수강생용 문서: [WEEK3_VUE_STUDY_MAKE_TODO_LIST_APP_STUDENT.md](week3-2/docs/handbook/WEEK3_VUE_STUDY_MAKE_TODO_LIST_APP_STUDENT.md)
- 강사용 문서: [todo-list-app/README.md](week3-2/src/todo-list-app/README.md)

## 공통 학습 목표

- Vue 프로젝트 생성 흐름과 기본 파일 구조를 이해한다.
- Todo List 앱을 만들며 Vue 핵심 기능을 단계적으로 익힌다.
- 상태 기반 UI 사고방식으로 화면 동작을 설명할 수 있다.

## 공통 진행 흐름

1. 프로젝트 세팅, PrimeVue 최신 버전 설치, 파일 구조 확인
2. 입력 기능 구현과 상태 바인딩 이해
3. 목록 렌더링과 반복 구조 이해
4. 완료 토글과 조건부 표현 이해
5. 삭제 기능과 상태 갱신 이해
6. 필터링과 계산된 값 이해
7. 최종 동작 점검 및 확장 과제 안내

- 실습 후 스스로 점검할 항목이 무엇인지

### 강사용 문서

- 어떤 순서로 설명할지

## 운영 원칙

- 한 번에 많은 문법을 설명하지 않는다.
- 기능 구현 직후 동작을 확인하고 상태 흐름을 다시 정리한다.
- 수강생은 결과물을 따라가며 개념을 익히고, 강사는 상태 중심 사고를 반복해서 강조한다.

- 할 일 입력 가능
- 할 일 추가 가능
- 목록 출력 가능
- 완료 토글 가능
- 삭제 가능
- 빈 목록 메시지 표시 가능
- 필터링 가능
- 남은 할 일 개수 표시 가능
